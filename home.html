lockscreen
2024		 Work in progress. A simple lock screen for Wayland that listens for the D-Bus "Lock" message from org.freedesktop.login1, using the libdbus low-level C API. lockscreen then uses the Wayland protocol to interact with the Wayland display and lock the session.
